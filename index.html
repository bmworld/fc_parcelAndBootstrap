BABEL- 구형브라우저에 최신JS문법으로 Compile

(ES6 ~ ES8) -> ES5

$ npm i -D @babel/core @babel/preset-env

바벨 설정파일 생성 (.bablerc.js)

module.exports =
{ presets: ['@babel/preset-env']}

설정완료 => 이제 모든 Javascript코드는 babel을 통해서 ES5문법으로 변환된다.

Parcel 은 모듈 안에서 설정 파일(예: .babelrc, .postcssrc)을 발견했을 때 자동으로 이 변환을 실행합니다.

Bable은 package.js 속에 browerslist에 영향을 받아 브라우저마다 최적화하여 변환한다. ==> 필수 입력하시라
