Autoprefixer:브라우저마다 지원하는 CSS코드 자동 생성 Plugin

https://github.com/postcss/autoprefixer

DevDependency로 설치한다

$ npm i -D postcss autoprefixer

postcss와 postcss 플러그인 autoprefixer설치

Parcel 은 모듈 안에서 설정 파일(예: .babelrc, .postcssrc)을 발견했을 때 자동으로 이 변환을 실행합니다.

package.json 파일 설정 추가 > json속성에 추가된 것 브라우저일 경우, CSS에 해당 코드 자동으로 추가해줌

    browserslist (중간에 S 빼먹지 않게 조심)
  • "> 1%" : 브라우저 점유율
  • "last 2 versions" : 브라우저의 최근 2개 버전

.postcssrc.js > 파일이름앞에 .을 붙이면 ? 구성옵션 or 숨김파일을 의미

rc: Runtime Configuration의 약어 => 구성파일을 의미한다.
